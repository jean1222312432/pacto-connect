# API Reference

The API reference is generated automatically from the TypeScript source files. Run `npm run generate:api-docs` in `apps/docs` to regenerate it, or push a change to `packages/*/src` — CI will keep it in sync.

<Callout type="info">
  All types are exported from their respective package root. You never need to import from internal paths.
</Callout>
